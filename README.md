# 📘 Technical Writing & QA Documentation Portfolio

This repository shows structured technical documentation supporting Software Quality Assurance processes across web, mobile, desktop and OTT platforms.

It demonstrates practical experiences in:

- Test Planning & Strategy Development
- End-to-End QA Workflow Documentation
- Device Setup & Cross-Platform Validation
