# Programming Tutorials for QA Professionals

This section provides beginner-friendly programming documentation tailored specifically for Quality Assurance professionals.

The goal is not to teach full-scale software development, but to equip QA engineers with practical coding skills needed for:

- Test automation
- API validation
- Data validation
- Log analysis
- Debugging
- Database verification

---

## 📘 Included Tutorials

### JavaScript for QA
Covers foundational JavaScript concepts commonly used in web automation and testing frameworks.

### TypeScript for QA
Introduces TypeScript fundamentals with emphasis on structured test automation and type safety.

### Python for QA
Focuses on scripting, API validation, data parsing, and automation basics.

### SQL for QA
Covers database validation, data integrity checks, joins, filtering, and aggregation queries.

---

## 🎯 Purpose

These tutorials demonstrate:

- Programming awareness for testers
- Automation readiness
- Backend validation capability
- API and database verification knowledge
- Practical, test-focused coding skills

This section reflects a QA professional who understands both testing strategy and technical implementation.
